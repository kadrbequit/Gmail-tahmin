# Gmail Profil Tahmin Aracı

Gmail adresinden **ad**, **soyad** ve **şehir** tahmini yapan Python aracı.

## 🚀 Özellikler

- ✅ Gmail adresinden ad/soyad çıkarımı
- ✅ Genderize.io API ile cinsiyet tahmini
- ✅ ip-api.com ile IP'den şehir sorgusu
- ✅ Domain bazlı şehir fallback
- ✅ Toplu email sorgusu
- ✅ JSON çıktı kaydetme

## 📦 Kurulum

```bash
# 1. Python ve pip kur (Linux/Mac/Termux)
pip install requests

# 2. Projeyi klonla
git clone  https://github.com/kadrbequit/Gmail-tahmin.git
cd Gmail-tahmin

# 3. Çalıştır
python gmail_tahmin.py
